# 🏛️ Orbyte - Smart Attendance System for Modern Campuses

Orbyte is a comprehensive digital attendance solution designed to streamline real-time tracking for educational institutions. This platform provides a seamless experience for lecturers, students, and administrators to manage academic presence with smart insights and instant reporting.

## 🚀 Key Features
- **Multi-Role Access:** Dedicated interfaces and permissions for **Lecturers, Admins, and Students**.
- **Real-time Tracking:** Instant check-in and attendance monitoring.
- **Data Export:** Integrated with `xlsx.min.js` to allow easy export of attendance reports to CSV and Excel formats.
- **PWA Ready:** Includes a Service Worker and Web Manifest for an app-like experience on mobile devices.
- **Secure Authentication:** Professional login system with role-based validation.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
- **Libraries:** [SheetJS (xlsx.min.js)](https://github.com/SheetJS/sheetjs) for data processing.
- **Features:** PWA (Progressive Web App) for offline capabilities and mobile installation.

## 📁 File Structure Highlights
- `index.html` : The main entry point & professional login portal.
- `script.js` : Core logic for role-based navigation and authentication.
- `style.css` : Modern UI styling with a focus on usability and aesthetics.
- `serviceworker.js` & `manifest.json` : Progressive Web App configuration.
- `xlsx.min.js` : Library for handling Excel/Spreadsheet exports.
